Quick Help

This app can be used as a launchpad mini mk3 style midi controller, it does not play sound itself. It is meant to let you explore if you like the launchpad workflow. If you do please by a Novation Launchpad Mini [MK3] through this partner link to support my site!

When using the app on an Android device connect it via usb to your computer, swipe down the top menu and select the usb options. There select 'MIDI'.

When using the app for the first time, choose the midi in/out devices in the 'Settings' tab.

So far it only has been tested in conjunction with DrivenByMoss and Bitwig Studio. Read more about how to use it here.
